# 欢迎

这是一个轻量级的 XP 主题博客原型，面向静态托管（GitHub Pages），专注内容展示，不包含评论或推荐等社交功能。

## 已实现

- 桌面文件夹可打开资源管理器。
- Markdown 文件可在阅读器中打开。
- 任务栏可切换字体。

## 下一步想法

- 在 `content/` 下继续添加文件夹或文章。
- 替换 `images/wallpaper.jpg` 作为桌面壁纸。
