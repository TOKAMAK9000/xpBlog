# 这是什么

这是一个博客项目。

我希望这个项目能做到：

1. 简单可维护。有非常清晰的项目结构，实现一个小而美的博客系统，我们除了需要把页面本身做得好看一些以外，不需要任何评论，批注，推荐等等复杂功能。
2. 无需传统的后端概念（无服务器资源），即可直接依托于 github pages
部署。

界面要求：
1. 完全仿照Windows XP系统。需要一比一复刻（当然，任务栏里的按钮都**不需要**可以点击）。我为这个页面提供了一个壁纸，放在images/wallpaper.jpg
2. 可以点击桌面上的文件夹，以打开文件管理器，里面可能包含子文件夹，也可能包含文章。

除此之外，只要能够正常显示markdown即可。要允许我指定字体。
